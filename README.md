# 🚴 Bike Purchase Data Analysis (Excel Project)

## 📌 Project Overview
This project analyzes a **Bike Purchase dataset** using **Microsoft Excel**.

The goal of this project was to clean the data, transform categorical variables, create pivot tables, and build an interactive dashboard to uncover customer purchasing behavior patterns.

---

# 🧹 Data Cleaning & Preparation

## 1️⃣ Created a Working Sheet
- Duplicated the original dataset
- Performed all cleaning and transformations on a separate working sheet to preserve raw data integrity

---

## 2️⃣ Removed Duplicates
- Used **Excel → Data → Remove Duplicates**
- Ensured no repeated customer records remained

---

## 3️⃣ Standardized Abbreviations

Converted abbreviated values into full descriptive labels:

- **M → Male**
- **F → Female**

- **M → Married**
- **S → Single**

This improved readability and clarity in analysis and dashboard visuals.

---

## 4️⃣ Data Transformation & Categorization

### ✔ Age Group Categorization

Created a new column to categorize age into:

- **Young**
- **Middle Aged**
- **Old**

This allowed easier comparison of purchase patterns across life stages.

---

### ✔ Income Level Categorization

Grouped income into three categories:

- **Lower Income**
- **Middle Income**
- **High Income**

This helped identify purchasing power trends.


---

# 📊 Pivot Tables & Analysis

Pivot tables were created to summarize and analyze bike purchase behavior across different demographics.

### 🚴 Bikes Purchased by Gender
- Compared total purchases between Male and Female customers.

### 👥 Bikes Purchased by Age Group
- Analyzed which age category purchases more bikes.

### 💰 Bikes Purchased by Income Level
- Identified trends in bike purchases across income categories.

### 🚗 Bikes Purchased vs Number of Cars Owned
- Examined the relationship between car ownership and bike purchases.

---

# 📈 Dashboard Development

An interactive Excel dashboard was created using pivot charts and slicers.

## 📊 Charts Included

- Clustered Column Chart: **Bike Purchases by Gender**
- Clustered Column Chart: **Bike Purchases by Age Group**
- Stacked Column Chart: **Bike Purchases by Income Level and Number of Cars Owned**

---

## 🎛 Interactive Slicers Added

The dashboard includes slicers for:

- Marital Status  
- Income Level  
- Home Ownership  
- Commute Distance  

These slicers allow dynamic filtering of the entire dashboard for deeper insights.

---


# 📌 Key Insights

- Purchase behavior varies significantly by age group, Middle aged customers buy or even view the product significantly more.
- Income level strongly influences bike purchases, high income customers are less likely to buy.
- Car ownership impacts the likelihood of purchasing a bike, customers with 4 cars are least likely to buy, customers with 1 car are less likely and with 2 or 3 cars, buy more.
- Dashboard slicers allow multi-dimensional filtering for better business insights.

---

# 📎 Conclusion

This project demonstrates a complete Excel-based data analysis workflow:

- Data Cleaning  
- Data Transformation  
- Pivot Table Analysis  
- Interactive Dashboard Creation  

It highlights how Excel can be used effectively to analyze customer behavior and present insights visually through dashboards.
